# Adspirer Documentation

Documentation for [Adspirer](https://www.adspirer.com) — the MCP server that connects AI assistants to advertising platforms.

**Live:** [www.adspirer.com/docs](https://www.adspirer.com/docs)

## What's Here

24 MDX pages covering:

- **Getting Started** — Introduction and quickstart guide
- **AI Clients** (8) — ChatGPT, Claude, Claude Code, Cursor, Codex, OpenClaw, Windsurf, Custom GPT
- **Ad Platforms** (4) — Google Ads (39 tools), Meta Ads (20), LinkedIn Ads (28), TikTok Ads (4)
- **Agent Skills** (3) — Overview, tool catalog (100+ tools), core workflows
- **Guides** (7) — SEO pillar content for Google Ads, Meta Ads, TikTok, and AI advertising

## Architecture

Hosted on [Mintlify](https://mintlify.com) and served at `www.adspirer.com/docs` via Vercel rewrites. The main website (`website-v3-claude-code` repo) proxies `/docs/*` requests to `adspirer.mintlify.dev`.

```
User → www.adspirer.com/docs/* → Vercel rewrite → adspirer.mintlify.dev/docs/*
```

### Auto-generated by Mintlify

| Endpoint | Purpose |
|----------|---------|
| `/docs/llms.txt` | Documentation index for LLMs |
| `/docs/llms-full.txt` | Full documentation content for LLMs |
| `/docs/sitemap.xml` | Sitemap with all page URLs |
| `/docs/mcp` | MCP server for doc search |
| `/docs/skill.md` | Agent skill definition |
| `/docs/.well-known/skills/index.json` | Skills discovery index |
| Any page + `.md` | Markdown content negotiation |

## Development

Install the [Mintlify CLI](https://www.npmjs.com/package/mint):

```bash
npm i -g mint
```

Preview locally:

```bash
mint dev
```

## Publishing

Push to `main` — Mintlify auto-deploys via GitHub App.
