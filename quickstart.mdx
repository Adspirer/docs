<Steps>
  <Step title="Sign Up">
    Create your free Adspirer account at [adspirer.ai](https://adspirer.ai?utm_source=docs&utm_medium=page&utm_content=signup). No credit card required — free tier includes 15 tool calls per month.
  </Step>

  <Step title="Connect Your AI Client">
    Pick your AI assistant and follow the setup guide:

    **Chat Apps**

    <CardGroup cols={2}>
      <Card title="ChatGPT" icon="/icons/chatgpt.svg" href="/ai-clients/chatgpt">
        Add the MCP connector in ChatGPT Settings. Requires Plus or Pro.
      </Card>
      <Card title="Claude" icon="/icons/claude.svg" href="/ai-clients/claude">
        Add the MCP connector in Claude Settings. Requires Pro, Max, Team, or Enterprise.
      </Card>
    </CardGroup>

    **Developer Tools**

    <CardGroup cols={2}>
      <Card title="Claude Code" icon="/icons/claude-code.svg" href="/ai-clients/claude-code">
        Plugin install or one MCP command from your terminal.
      </Card>
      <Card title="Cursor" icon="/icons/cursor.svg" href="/ai-clients/cursor">
        One-command installer with 5 skills and subagent.
      </Card>
      <Card title="Codex" icon="/icons/codex.svg" href="/ai-clients/codex">
        One-command installer for autonomous agent setup.
      </Card>
      <Card title="OpenClaw" icon="/icons/openclaw.svg" href="/ai-clients/openclaw">
        Plugin install with 14KB skill file built in.
      </Card>
      <Card title="Windsurf" icon="/icons/windsurf.svg" href="/ai-clients/windsurf">
        Add MCP config to Windsurf settings.
      </Card>
      <Card title="Custom GPT" icon="/icons/chatgpt.svg" href="/ai-clients/custom-gpt">
        Use Adspirer inside ChatGPT via the GPT Store.
      </Card>
    </CardGroup>

    **Quick MCP Config Reference**

    <Tabs>
      <Tab title="Claude Code">
        ```bash
        claude mcp add --transport http adspirer https://mcp.adspirer.com/mcp
        ```
        Or use the plugin system: `/plugin marketplace add amekala/ads-mcp` then `/plugin install adspirer`
      </Tab>
      <Tab title="Cursor">
        ```bash
        bash <(curl -fsSL https://raw.githubusercontent.com/amekala/ads-mcp/main/plugins/cursor/adspirer/install.sh)
        ```
        Run from your **system terminal**, not Cursor's built-in terminal. Then restart Cursor.
      </Tab>
      <Tab title="Codex">
        ```bash
        bash <(curl -fsSL https://raw.githubusercontent.com/amekala/ads-mcp/main/plugins/codex/adspirer/install.sh)
        ```
        Then run `codex mcp login adspirer` to authenticate.
      </Tab>
      <Tab title="OpenClaw">
        ```bash
        openclaw plugins install openclaw-adspirer
        openclaw adspirer login
        ```
      </Tab>
      <Tab title="Windsurf">
        Add to `~/.codeium/windsurf/mcp_config.json`:
        ```json
        {
          "mcpServers": {
            "adspirer": {
              "type": "streamable-http",
              "url": "https://mcp.adspirer.com/mcp"
            }
          }
        }
        ```
      </Tab>
      <Tab title="ChatGPT / Claude">
        No config needed — add the MCP connector from your app's Settings page:

        **MCP Server URL:** `https://mcp.adspirer.com/mcp`
      </Tab>
    </Tabs>
  </Step>

  <Step title="Try It Out">
    Once connected, ask your AI assistant:

    <Prompt description="Verify your ad platform connections are working." actions={["copy", "cursor"]}>
    Check my connected ad platforms
    </Prompt>

    You should see a table showing your connected accounts with IDs, names, and connection status.

    Then try something real:

    <Prompt description="Research keywords with real CPC data from Google Keyword Planner." actions={["copy", "cursor"]}>
    Research Google Ads keywords for "project management software" — show search volume, CPC, and competition level
    </Prompt>
